# 카카오 알림톡 셋업 가이드 (처음부터)

> 알림톡은 **사전 준비 3단계 + 검수 승인(영업일 소요)** 이 끝나야 보낼 수 있습니다.
> 아래를 순서대로 하시고, 마지막에 나오는 **pfId(발신프로필 ID)** 와 **templateId(템플릿 ID)** 두 개만 저에게 주시면 코드/대시보드에 바로 연결합니다.

---

## 1단계 — 카카오톡 채널 만들기 (없으면)
1. **카카오톡 채널 관리자센터** (`business.kakao.com`) 접속 → 채널 개설
2. 채널 이름: 예) `김가경 국어 연구소` / `[오름] 국어`
3. 채널을 **공개 + 검색 허용** 상태로 (검수에 필요)
4. (권장) 비즈니스 인증 = **사업자등록증**으로 비즈니스 채널 인증 → 발송 한도·신뢰도↑

## 2단계 — Solapi에 카카오 채널 연동 → 발신프로필(pfId)
1. Solapi 콘솔 → **카카오 → 카카오 채널 연동(발신프로필 등록)**
2. 채널 **검색용 ID**(@아이디) 입력 → 채널 관리자 인증(카카오 인증)
3. 연동 완료 시 **발신프로필 ID = `pfId`** 가 발급됨 ← **이거 저장**

## 3단계 — 알림톡 템플릿 등록 + 검수 요청
1. Solapi 콘솔 → **카카오 → 알림톡 템플릿 → 새 템플릿**
2. 아래 문구를 등록 (변수는 `#{이름}` 형식 그대로) → **검수 요청**
3. 승인까지 보통 **1~2 영업일**. 승인되면 **`templateId`** 발급 ← **이거 저장**
4. ⚠️ **정보성만 가능**(과제·성적 안내 OK / 광고·홍보 문구 ❌). 아래 문구는 정보성이라 통과 잘 됩니다.

### 제출용 템플릿 문구 (그대로 복사)

**[템플릿 A] 미제출·미완 독려**
```
[김가경 국어 연구소]
안녕하세요, #{이름} 학생 학부모님.
이번 주(#{주차}) 과제 완수율이 #{완수율}입니다.
약속한 기한까지 마무리할 수 있도록 가정에서 함께 살펴봐 주시면 감사하겠습니다.

문의: #{선생님}
```

**[템플릿 B] 주간 과제 결과 요약**
```
[김가경 국어 연구소]
#{이름} 학생의 이번 주(#{주차}) 과제 결과를 안내드립니다.
· 평균 정답률: #{정답률}
· 완수율: #{완수율}
자세한 내용은 가정통신문을 참고해 주세요.

문의: #{선생님}
```
> 변수명(`이름`, `주차`, `완수율`, `정답률`, `선생님`)은 **이 철자 그대로** 등록하세요. 코드가 같은 이름으로 채워 보냅니다.

---

## 4단계 — 저에게 전달할 것
승인 완료되면 아래 2개만 알려주세요 (채팅 또는 스크립트 속성에 직접 입력):
- **pfId**: (발신프로필 ID)
- **templateId**: 템플릿 A / 템플릿 B 각각의 ID

→ 그러면 대시보드 시나리오에 **"알림톡 우선 → 실패 시 문자 자동 대체"** 로 연결해 드립니다.
(코드에는 이미 알림톡 폴백 구조가 들어가 있어, pfId·templateId만 꽂으면 됩니다.)

---

## 비용·도달 참고
- 알림톡 ≈ 7~15원(문자보다 저렴), 스팸 차단 없음, 도달률 높음
- 미설치/수신거부 등 실패 시 **문자(LMS)로 자동 대체** 발송되도록 구성
- 정보성이라 야간발송 제한·수신거부 의무 대상 아님 (홍보 문구 섞으면 광고로 전환되니 금지)
